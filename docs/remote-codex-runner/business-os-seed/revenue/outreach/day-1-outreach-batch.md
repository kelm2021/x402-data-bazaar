## Message Rules

- one fixed-fee offer per message
- ask for function + bottleneck + next-week outcome
- no feature dump, no tool pitch
- keep the message short enough for contact forms

## Core Message Template

```text
Hi {{name}}, I help founders/operators install a working execution system for one business function in 5 business days.

If useful, send:
- function to fix
- current bottleneck
- outcome needed by next week

I can return scope and timeline the same day.
```
